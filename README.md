# Public Notes

By default, all of my notes live in a private location.

The script [publish-notes.ps1](https://github.com/imJoshuaRice/nvim-config/blob/main/scripts/publish-notes.ps1) (triggered by `Space gp`) scans for the presence of `public: true` in note frontmatter and then publishes it to this repository.

The folder structure is also mirrored.
